# Backend Node.js Starter

## Run in development

```bash
npm install
npm run dev
```

## Run in production mode

```bash
npm start
```

## Health check

- Endpoint: `GET /api/health`

## Swagger API docs

- URL: `http://localhost:4000/api-docs`
- Dung de test API truc tiep tren giao dien Swagger UI.
